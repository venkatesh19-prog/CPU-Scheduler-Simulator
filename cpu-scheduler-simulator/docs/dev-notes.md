# Developer Notes

## Project Overview
This project simulates CPU Scheduling Algorithms (FCFS, SRTF, RR, Priority) and visualizes results using a Gantt chart.

## Files & Structure
- `app.py` – Main Flask backend
- `static/js/app.js` – Frontend logic
- `static/css/style.css` – UI design
- `templates/index.html` – Main web interface

## Future Improvements
- Add process editing
- Add preemptive priority scheduling
- Add export options for Gantt chart
- Support dark mode

## Notes for Developers
Please keep UI modular and backend algorithms clean.
